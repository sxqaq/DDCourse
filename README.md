# DDCourse

[English](./README.en.md) · 简体中文

> 专为本地课程设计的 Learning Player。

DDCourse 是一款轻量、本地优先的学习播放器。选择课程文件夹后，它会自动整理章节、记录进度、收藏重点和课程笔记，并在下次打开时继续播放。视频只在本地读取，不会上传到服务器。

## AI 创作声明

本项目的全部工程内容，包括产品设计、源代码、测试、构建与发布配置、使用文档以及本 README，均由 GPT-5.6 制作。

## 功能特性

- 选择本地文件夹或多个视频文件
- 按子文件夹自动整理课程合集
- 自动记录播放位置，并支持断点续播
- 为每个视频单独记忆播放速度
- 展示课程完成度与本周专注时长
- 搜索课程及“仅看未完成”筛选
- 支持 0.5×–3× 播放速度
- 基于 Web Audio API 的人声增强
- 上一节、下一节及前后 10 秒跳转
- 键盘快捷键与移动端响应式布局
- 以 JSON 文件导入、导出学习进度
- 拖拽载入本地视频
- 支持安装为桌面 PWA 应用

## 安装应用

### Windows EXE 安装包（推荐）

从项目 Release 页面下载 `DDCourse-Setup-1.2.1.exe`，双击后按照向导完成安装。安装器支持：

- 自定义安装位置
- 自动创建桌面快捷方式
- 创建开始菜单入口
- 中文与英文安装界面
- 通过系统“已安装的应用”正常卸载

如需自己生成安装包：

```bash
npm install
npm run desktop:build
```

生成的安装程序位于 `release/DDCourse-Setup-1.2.1.exe`。

## 学习进度备份与恢复

进度保存在当前浏览器或桌面应用自己的本地存储中，不包含视频文件、笔记或收藏。

- 导出：点击课程库底部的“导出进度”，应用会下载 `DDCourse-progress-YYYY-MM-DD.json`。
- 导入：点击“导入进度”并选择此前导出的 JSON。导入内容会与本机已有记录合并；同一视频的导入记录会覆盖本机记录。
- 兼容范围：桌面版和 Web/PWA 版使用相同格式，可以相互迁移。视频需要由用户在目标设备上重新选择，且相对路径和文件大小应保持一致，应用才能匹配原记录。

备份文件格式版本为 `1`，结构示例如下：

```json
{
  "app": "DDCourse",
  "formatVersion": 1,
  "exportedAt": "2026-07-11T08:30:00.000Z",
  "progress": {
    "课程/01-导论.mp4::104857600": {
      "time": 125.4,
      "duration": 1800,
      "done": false,
      "updatedAt": "2026-07-11T08:29:50.000Z",
      "speed": 1.25
    }
  }
}
```

应用会拒绝其他应用、未知格式版本、负数时间或字段不完整的文件，避免无效数据污染现有进度。旧版 DDCourse 导出的无 `formatVersion` 文件仍可导入。

完整字段和手工编写方法参见 [学习进度 JSON 格式说明](./PROGRESS_FORMAT.md)。1.2.1 会自动迁移旧版包含“最后修改时间”的视频 ID。

### 浏览器安装

1. 使用最新版 Chrome 或 Microsoft Edge 打开已部署的 DDCourse。
2. 点击应用右上角的“安装应用”。
3. 确认安装后，DDCourse 会出现在桌面和开始菜单中，并以独立窗口运行。

如果没有看到安装按钮，可打开浏览器菜单，选择“安装 DDCourse”或“应用 → 将此站点安装为应用”。

### Web 版本地开发

需要 Node.js 22.13 或更新版本。

```bash
npm install
npm run dev
```

然后打开终端中显示的本地地址。开发环境中的安装能力取决于浏览器；正式部署在 HTTPS 地址后可完整使用 PWA 安装。

## 键盘快捷键

| 按键 | 功能 |
| --- | --- |
| `空格` | 播放 / 暂停 |
| `←` / `→` | 后退 / 前进 10 秒 |
| `P` / `[` | 上一节 |
| `N` / `]` | 下一节 |
| `Q` / `W` / `E` / `R` | 1× / 1.25× / 1.5× / 2× |

## 隐私说明

视频直接从你的设备读取，不会上传。学习进度保存在当前浏览器的本地存储中，你可以随时导出为 JSON 文件备份。

## 技术栈

React 19、TypeScript、Vinext 与原生 CSS。没有数据库、账号系统或重型客户端状态库。

界面优先使用小米 MiSans 字体。MiSans 为小米科技有限责任公司提供的免费商用字体，使用遵循其官方字体知识产权许可协议。
