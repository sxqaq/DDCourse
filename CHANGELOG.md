# 更新日志 / Changelog

本文件记录 DDCourse 各版本面向用户和维护者的重要变化。

## 1.5.6 — 2026-07-22

### 修复与改进

- 修复切换课程时延迟 `pause` 事件可能覆盖新课程进度的问题；浏览其他课程的笔记后可直接切换视频并跳到对应时间。
- 本地偏好增加运行时归一化，损坏的合集、隐藏项、折叠项和上次选择数据不再导致启动白屏。
- 隐藏课程仍参与统计、重置与 Markdown 报告；笔记全文搜索可跨合集打开结果。
- 课程分组改为线性处理，超大课程列表使用虚拟窗口，缩略图缓存限制为 60 项 LRU。
- 桌面更新保持完全手动，退出时不会静默安装；UNC 路径复制改走受信 IPC 并保留服务器名称。
- 发布流程校验标签版本与 `main` 分支归属，Windows CI 覆盖基线版本到当前版本的数据持久性升级。
- 修复更新元数据、CSS 与图片处理依赖的生产漏洞；`npm audit --omit=dev` 为 0。

### 验证

- 32 项单元/SSR 测试、14 项浏览器端到端测试、真实媒体/字幕/离线启动、桌面升级持久性、安装包内容与打包版启动验证均通过。

### 1.5.5 遗漏补记

- 1.5.5 的应用内更新从设计上就是“检查、下载、安装”三步均由用户确认；1.5.6 进一步关闭退出时自动安装，补齐了此前更新日志没有明确写出的行为边界。
- 1.5.5 已加入跨视频笔记浏览、外挂字幕、画中画、视频全屏和缩略图预览；1.5.6 为这些路径补上真实媒体端到端覆盖，并允许从浏览模式直接跳转到对应视频时间。
- 1.5.5 已保留课程进度、文件夹、合集偏好、自定义字体与桌面笔记；1.5.6 新增基线版本到当前版本共用用户目录的升级测试，正式验证这些数据不会因升级丢失。

## 1.5.5 — 2026-07-22

### 新增

- Windows 安装版新增应用内手动更新：检查新版本、手动下载、显示进度，并在用户确认后重启安装。更新只替换程序文件，不清除课程进度、最近文件夹、折叠/主题等本地设置，也不清除“文档\\DDCourse”中的笔记。
- 合集右键菜单新增重命名、重置进度、文件管理器定位、Markdown 学习报告、置顶、暂不学习和收起；“暂不学习”的合集不计入全库进度。
- 视频右键菜单新增手动完成/未完成、单节重置、笔记浏览、插队下一节、软隐藏、复制路径和文件管理器定位。
- 支持同目录同名 `.srt` / `.vtt` 外挂字幕（含 UTF-8 与 GBK）、画中画、独立视频全屏、缩略图预览进度条、笔记与收藏全文搜索、合集 Markdown 学习报告，以及跟随系统/浅色/深色主题。

### 修复

- 修复播放过视频后合集无法切换的问题；上次播放位置现在只恢复一次。
- 修复空格在按钮聚焦时同时触发快捷键和默认点击，造成播放后立即暂停的问题。
- 修复字体设置被全局固定字体覆盖，以及视频遮挡字体面板的问题。
- 区分“视频全屏”和“应用全屏”，并在 Electron 中仅放行全屏权限。
- 手动完成状态加入持久覆盖标记，后续自动保存不会把用户选择改回去；重置或隐藏当前课程前会先停止播放器并保存。

### 发布与清理

- Release 同时发布安装程序、`latest.yml` 和 blockmap，供应用内更新安全校验与差分下载。
- 移除未使用的私有网站托管配置、Cloudflare/Sites 代码和依赖；不再发布官方 Web 站点。

## 1.5.0 — 2026-07-20

### 改进

- 课程合集改为每行 3 个、最多占用两行的紧凑网格，超过 6 个时在合集区域内滚动，不再挤压课程视频列表。
- 新增“收起已完成”操作；也可以右键任意课程合集选择“收起此合集”。已收起合集保存在本机，并可从紧凑入口随时恢复。
- 增加课程合集布局、批量收起、右键收起和恢复的端到端测试。

## 1.4.0 — 2026-07-15

### 新增

- 桌面笔记增加 `updatedAt`、删除墓碑、运行时 schema 校验、400ms 防抖保存和串行写入队列。
- 新增完整页面 SSR 测试、播放器停止生命周期 E2E、真实离线重新启动测试，以及 Electron 开发版和安装版冒烟测试。
- 新增安装包内容校验，确保 Electron 运行时所需的笔记 schema 和桌面资源均已打包。

### 改进

- 统一播放器停止生命周期。切换文件、目录或合集，以及刷新、重置和卸载时，会保存进度、暂停媒体、清空 `src` 并撤销 object URL。
- 将课程库、侧边栏、播放器舞台和笔记面板拆分为独立组件与 hooks，减少页面耦合。
- Service Worker 改为依据构建产物生成预缓存清单，提升正式环境离线启动的稳定性。
- 浏览器 API 改为 SSR 安全访问；本地存储不可用时回退到内存状态。
- 进度合并与笔记同步统一按时间戳判断新旧数据。

### 修复

- 修复 Windows 安装包遗漏 `app/notes-schema.mjs`，导致安装版可能无法读取或保存桌面笔记的问题。
- 修复损坏的本地进度对象可能阻止应用启动的问题。
- 修复异常播放速度、超大进度备份和超大自定义字体缺少限制的问题。

### 安全与发布

- Electron IPC 仅接受打包页面调用，并阻止窗口导航到非应用页面。
- 外部链接仅允许 HTTPS，Electron 权限请求默认拒绝。
- 笔记数量、标识符、路径和文本长度增加合理上限。
- GitHub Actions 构建阶段改为只读权限，发布权限移至独立任务，并锁定第三方 Action 的提交版本。
- Windows 安装包目前未配置代码签名证书，安装时可能显示“未知发布者”。

### 验证

- 单元与 SSR 测试：17 项通过。
- Playwright E2E：4 项通过，包括真实离线启动和播放器停止生命周期。
- Web 构建、Electron 开发版启动、Windows 安装包构建和安装版启动验证通过。
