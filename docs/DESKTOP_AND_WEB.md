# Web、PWA 与桌面端

Web/PWA 和 Electron 共用主要 React 界面，但文件权限不同。

| 能力 | Web/PWA | Electron |
| --- | --- | --- |
| 读取视频 | 用户选择的 File 对象 | `file:` URL |
| 记住目录 | 浏览器通常不保留目录句柄 | 记录最后目录路径 |
| 刷新目录 | 重新选择并授权同一目录 | 直接异步重新扫描 |
| 笔记主文件 | 无 | 文档目录 JSON |
| 离线外壳 | Service Worker | 桌面资源包 |

浏览器端“刷新”需要再次授权是浏览器权限模型的结果。桌面端通过 preload 的有限 IPC API 操作目录；渲染页面不能直接访问 Node.js 或任意文件系统接口。
