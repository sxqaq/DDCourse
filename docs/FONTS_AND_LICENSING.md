# 字体、导入与版权说明

DDCourse 默认优先使用 MiSans；如果设备没有安装 MiSans，会依次回退到 HarmonyOS Sans、OPPO Sans、思源黑体、苹方和系统无衬线字体。由于 MiSans 官方协议禁止把字体文件作为独立资源再次分发，公开源码仓库不附带裸字体文件。

MiSans 由小米科技有限责任公司提供。官方说明允许免费商用和软件嵌入，但要求软件明确注明使用 MiSans，禁止修改、二次开发字体，也禁止单独再分发或售卖字体文件。请以[小米官方许可协议](https://hyperos.mi.com/font/zh/download/)的最新内容为准。

## 用户自定义字体

“Aa 显示”支持以下文件：

- TrueType：`.ttf`
- OpenType：`.otf`
- Web Open Font：`.woff`、`.woff2`

导入文件保存在当前浏览器或 Electron 用户配置的 IndexedDB 中，只供本机 DDCourse 渲染使用，不会上传。用户可以从任何合法来源自行下载字体，并不局限于 MiSans。

用户有责任确认所选字体是否允许其具体用途，包括个人使用、商业使用、软件嵌入、网络字体使用及再分发。DDCourse 对第三方字体不提供、转让或暗示任何许可，也不对用户未经授权使用字体承担责任。

源码中的 `public/fonts/` 是字体管理与说明目录，`.gitignore` 默认排除用户放入的字体二进制，防止误提交第三方受版权保护的文件。

## 字号设置

字号提供“小、标准、大”三个档位。应用按整体比例缩放布局和文字，从而尽量保持侧栏、播放器与控件的相对尺寸，避免只放大文字造成截断或布局错位。设置保存在本机。
